About QueryPath Examples

The scripts provided in this package are intended to illustrate how QueryPath can work in various circumstances. Each example is self-contained, and makes use only of QueryPath and the core extensions.

You will find much more information about these examples on the wiki. There, each example is explained in detail, and example output is also shown.

Where to begin

If you are new to QueryPath, you may wish to start with simple_example.php, and then move on to xml.php and html.php.

As you click on each page, check out the "source code for this file" link. That will take you to a syntax-highlighted version of the source code. In most cases, the source code is under 50 lines and includes many comments.
