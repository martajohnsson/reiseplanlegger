			static mixed  
			$regex
			 =  '/&([\w]+);|&#([\d]+);|&#(x[0-9a-fA-F]+);|(&)/m'			[line 1315]
		
	


	
This is three regexes wrapped into 1. The | divides them.



1: Match any char-based entity. This will go in $matches[1]  2: Match any num-based entity. This will go in $matches[2]  3: Match any hex-based entry. This will go in $matches[3]  4: Match any ampersand that is not an entity. This goes in $matches[4]     This last rule will only match if one of the previous two has not already     matched.
