Class BottomUpCssEventHandler

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Implements interfaces:

  • CssEventHandler

Use a bottom-up searching algorithm.

This collects all of the events from a CssEventHandler, and then searches in reverse order, narrowing as it goes. Searching this way eliminates the need for duplicate checks.
