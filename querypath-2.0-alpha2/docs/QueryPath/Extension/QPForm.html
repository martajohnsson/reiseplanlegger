Create a new fieldset form item.



This will automatically select the fieldset as the active item. So  a call to $form->fieldset('Test')->textfield('Foo') will place the  textfield inside of the fieldset.

			Parameters:

			

					

				string  
				
				$legend: 
				
				 The legend (title, description) for this fieldset.			

					

				array  
				
				$options: 
				
				 Currently supported options: 
  • 'attributes': Associative array of HTML attributes.
